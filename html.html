→ <h1>, <h2>, <h3>,... : Tags de cabeçalho para títulos e subtítulos.

→ <P> : Paragrafo onde escreve o texto.

→ <img> : Exibe uma imagem na página.

→ <div> : Cria um bloco de conteúdo genérico que pode ser estilizado com CSS.

→ <a> : Cria um link para outra página.

→ <li> : Define um item de lista dentro de <ul> ou <ol>.

→ <br> : Insere uma quebra de linha.

→ <hr> :Insere uma linha horizontal.

→ <span> Define uma parte de texto que pode ser estilizada com CSS.
