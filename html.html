→ <p> : Define um paragrafo de texto.

→ <head> : Contém metadados sobre o documento. como título, links para arquivos CSS, scripits, e outras configurações.

→ <html> : A tag raiz que contém todo o conteúdo HTML da página.

→ <title> : Define o título da pagina, que aparece na aba do navegador.

→ <body> : Contém todo o conteúdo visível da página, como textos, imgagens, links, etc.

→ <a> : Define um link para outra página ou recurso.
